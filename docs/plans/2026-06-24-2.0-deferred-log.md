## Deferred features (revisit before ship)
- **Phase 1 — on-device verification.** Customizable keyboard is built + 92 tests green, but
  drag-reorder / add-remove / live keyboard re-render are NOT device-verified. _[owner — needs phones]_
- **Phase 1 — follow-ups.** Advanced token dispatch (cursor/hide/tab/calc/snippet/pack/overlay
  currently render as `.blank` in `renderKind`); per-key colour/size/span + `keyScale`; label
  override (split `Item` display vs inserted value); layout export/import (JSON via ShareLink/fileImporter).
- **Currency conversion** (Units&Conversion pack) — needs live FX rates (network); `UnitConverter`
  is intentionally offline. Ship length/mass/temperature (+ any other static units) only.
- **International/Formatting pack** — full phone / IBAN / national-ID formatting is
  libphonenumber-class. Scope kept tight: locale decimal/grouping separators, percent, currency
  symbol, basic phone grouping. Full validation/parsing deferred.
- **Phase 3 — non-requested items.** Inline calculator is already implemented behind a flag;
  left as-is (owner did not request surfacing it in this push). Any other "supporting" ideas
  beyond the four requested (iCloud sync, cursor controls, smart pack defaults, result tape).
